****************************************
community.windows.win_iis_webapplication
****************************************

**Configures IIS web applications**



.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Creates, removes, and configures IIS web applications.

Examples
--------

.. code-block:: yaml

    - name: Add ACME webapplication on IIS.
      community.windows.win_iis_webapplication:
        name: api
        site: acme
        state: present
        physical_path: C:\apps\acme\api
